# Dermatomes

> A **dermatome** is the area of skin innervated by a single spinal nerve root. Knowledge of dermatomal maps allows clinicians to localize the level of spinal cord injury, disc herniation, or nerve root compression from the distribution of sensory symptoms.

## 🔗 Key Connections
- [Spinal Cord](./anatomyLink/Nervous System/Central Nervous System/Spinal Cord.md) — each spinal level has a corresponding dermatome
- [Vertebral Column](./anatomyLink/Skeletal System/Spine/Vertebral Column.md) — vertebral level correlates (approximately) with dermatome
- [Brachial Plexus](./anatomyLink/Nervous System/Peripheral Nervous System/Brachial Plexus.md) — C5–T1 roots supply the upper limb
- [Lumbar Plexus](./anatomyLink/Nervous System/Peripheral Nervous System/Lumbar Plexus.md) — L1–L4 roots supply the anterior lower limb
- [Sacral Plexus](./anatomyLink/Nervous System/Peripheral Nervous System/Sacral Plexus.md) — L4–S3 roots supply the posterior lower limb

## 🗂️ Key Dermatomal Landmarks

### Head & Neck
| Level | Area |
|-------|------|
| **CN V (V1/V2/V3)** | Entire face (not a spinal root) |
| **C2** | Back of scalp + upper neck |
| **C3** | Posterior neck; lower jaw area |
| **C4** | Shoulder cape (capes the shoulder) |

### Upper Limb (Brachial Plexus C5–T1)
| Level | Area | Reflex |
|-------|------|--------|
| **C5** | Lateral arm (deltoid patch) | Biceps (C5–C6) |
| **C6** | Lateral forearm + thumb + index finger | Brachioradialis (C6) |
| **C7** | Middle finger | Triceps (C7) |
| **C8** | Ring + little finger + medial forearm | — |
| **T1** | Medial arm | — |

### Trunk
| Level | Landmark |
|-------|----------|
| **T4** | Nipple line |
| **T6** | Xiphisternum |
| **T10** | Umbilicus |
| **T12** | Inguinal ligament |
| **L1** | Inguinal region |

### Lower Limb (Lumbar + Sacral)
| Level | Area | Reflex |
|-------|------|--------|
| **L1** | Groin/inguinal region | Cremasteric |
| **L2** | Anterior upper thigh | — |
| **L3** | Anterior lower thigh + knee | Knee (L3–L4) |
| **L4** | Medial lower leg + medial foot | Knee (L3–L4) |
| **L5** | Lateral lower leg + dorsum of foot + great toe | — |
| **S1** | Lateral foot + sole + little toe | Ankle (S1–S2) |
| **S2** | Posterior thigh |  |
| **S3–S5** | Perineum (saddle area) + perianal | Anal wink |

## 🔑 Clinically Important Dermatomes

### Disc Herniation Levels
| Disc | Root Compressed | Sensory Loss | Motor Weakness | Reflex Lost |
|------|----------------|--------------|----------------|-------------|
| **C5–C6** | C6 | Lateral forearm, thumb | Wrist extensors, biceps | Biceps, brachioradialis |
| **C6–C7** | C7 | Middle finger | Triceps, wrist flexors | Triceps |
| **L3–L4** | L4 | Medial shin | Quadriceps, dorsiflexion | Knee jerk |
| **L4–L5** | L5 | Dorsum foot, great toe | EHL, dorsiflexion | None (or posterior tibial) |
| **L5–S1** | S1 | Lateral foot, sole | Plantarflexion, eversion | Ankle jerk |

### Spinal Cord Injury Levels
| Level | Key Function Lost |
|-------|------------------|
| **C3–C5** | Diaphragm (phrenic nerve C3–5); ventilator dependent above C4 |
| **C5** | Shoulder abduction + elbow flexion |
| **C6** | Wrist extension (key for tenodesis grip) |
| **C7** | Elbow extension (transfer ability) |
| **T1** | Intrinsic hand muscles (fine hand function) |
| **T4** | Nipple level; good trunk stability |
| **T10** | Umbilicus; ambulatory potential |
| **L1–L2** | Hip flexors |
| **S2–S4** | Bladder, bowel, sexual function (sacral micturition center) |

### Cauda Equina vs Conus Medullaris
| | Conus Medullaris (S3–S5) | Cauda Equina (below L1) |
|--|--------------------------|------------------------|
| Lesion type | UMN + LMN mixed | Pure LMN |
| Bladder | Spastic (UMN) or flaccid | Flaccid (areflexic) |
| Saddle anesthesia | Yes | Yes |
| Emergency? | Yes | Yes |
