# 🌡️ Spinothalamic Tract (Anterolateral System)

> **The spinothalamic tract is the principal ascending pathway for pain, temperature, and crude touch — its fibres cross the midline within 1-2 spinal segments of entry, travelling in the contralateral anterolateral white matter to the thalamus and cortex.**

## 🔗 Key Connections
- [Somatosensory System](./anatomyLink/Nervous System/Neuro System (Lippincott)/02 - Sensory Systems/Somatosensory System.md) — one of the two major somatosensory pathways (alongside dorsal column–medial lemniscal)
- [Pain Pathways](./anatomyLink/Nervous System/Neuro System (Lippincott)/02 - Sensory Systems/Pain Pathways.md) — spinothalamic tract is the anatomical substrate of pain and temperature transmission
- [Spinal Cord Circuits](./anatomyLink/Nervous System/Neuro System (Lippincott)/02 - Sensory Systems/Spinal Cord Circuits.md) — dorsal horn laminae I, IV, V are the synaptic relay stations
- [Sensory Receptors](./anatomyLink/Nervous System/Neuro System (Lippincott)/02 - Sensory Systems/Sensory Receptors.md) — Aδ fibres (fast pain, cool) and C fibres (slow pain, warm) are the primary afferents
- [Thalamus](./anatomyLink/Nervous System/Neuro System (Lippincott)/04 - Brain Regions/Thalamus.md) — VPL (ventral posterolateral nucleus) receives contralateral body spinothalamic input
- [Cerebral Cortex](./anatomyLink/Nervous System/Neuro System (Lippincott)/04 - Brain Regions/Cerebral Cortex.md) — S1 (postcentral gyrus) and S2 (parietal operculum) for discriminative aspects; insula/ACC for affective
- [Brainstem](./anatomyLink/Nervous System/Central Nervous System/Brainstem.md) — spinoreticular and spinomesencephalic components relay to reticular formation and PAG
- [Neurotransmitters](./anatomyLink/Nervous System/Neuro System (Lippincott)/01 - Cellular Neuroscience/Neurotransmitters.md) — glutamate (fast), substance P (slow); opioids modulate at dorsal horn level

## 🗂️ Key Information

| Property | Detail |
|----------|--------|
| Modalities | Pain, temperature, crude/light touch, itch |
| Primary afferents | Aδ (fast, sharp pain; cool temperature) and C fibres (slow, burning pain; warm temperature) |
| Decussation | Anterior white commissure, **within 1-2 spinal segments** of entry (oblique course) |
| Location in cord | Contralateral anterolateral white matter |
| Thalamic relay | VPL nucleus (body); VPM (face, via trigeminothalamic tract) |
| Clinical hallmark | Lesion → **contralateral** pain/temperature loss **below lesion level** |

## 🔑 Three-Neuron Pathway

### 1st Order Neuron (Primary Afferent)
| Fibre | Myelination | Diameter | Speed | Modality |
|-------|------------|---------|-------|---------|
| Aδ | Thin myelin | 1-5 μm | 5-30 m/s | Fast/sharp pain; cold temperature |
| C | Unmyelinated | 0.2-1.5 μm | 0.5-2 m/s | Slow/burning pain; warm temperature; itch |

- Cell body: **dorsal root ganglion** (DRG)
- Enters via dorsal root → ascends/descends **1-2 levels** in **Lissauer's tract** (dorsolateral fasciculus) before synapsing
- Synapses in **dorsal horn** using glutamate (fast) + substance P / CGRP (slow, sustained)

### 2nd Order Neuron (Spinal)
| Lamina (Rexed) | Name | Input |
|----------------|------|-------|
| Lamina I | Marginal zone | Aδ and C fibres; nociceptive-specific and thermoreceptive neurons |
| Lamina II | Substantia gelatinosa | C fibres; enkephalinergic interneurons (pain modulation — gate control) |
| Lamina V | WDR neurons | Aδ; wide dynamic range neurons receiving somatic + visceral convergence (referred pain) |

- **Crosses midline** via **anterior white commissure** (1-2 segments above entry level)
- Ascends in contralateral **anterolateral column** (spinothalamic tract proper)

### 3rd Order Neuron (Thalamo-cortical)
- **VPL nucleus** → posterior limb of internal capsule → **primary somatosensory cortex (S1)**, postcentral gyrus, Brodmann areas 3, 1, 2
- Also projects to **S2** (parietal operculum) and **posterior insula** (interoception, pain affect)

## 🔑 Subdivisions of the Anterolateral System

| Tract | Target | Function |
|-------|--------|---------|
| **Neospinothalamic** | VPL thalamus → S1 cortex | Discriminative aspects: location, intensity, quality of pain/temperature |
| **Paleospinothalamic / Spinoreticular** | Brainstem reticular formation → intralaminar thalamus → ACC, insula | Affective-motivational component: suffering, arousal, autonomic response |
| **Spinomesencephalic** | Periaqueductal grey (PAG), superior colliculus | Descending pain modulation; orienting reflex |
| **Spinohypothalamic** | Hypothalamus | Neuroendocrine and autonomic response to pain |

## 🔑 Somatotopic Organisation

The spinothalamic tract is **somatotopically arranged** in the anterolateral column:
```
(most medial/anterior = sacral/lumbar)
(most lateral/posterior = cervical/thoracic)

Lamination (medial → lateral):
Sacral (S) — Lumbar (L) — Thoracic (T) — Cervical (C)
```

**Clinical relevance**: Syringomyelia (central cord cavity) → expands laterally → disrupts **crossing fibres in anterior white commissure** → bilateral cape/suspended pattern pain/temperature loss at the level of the lesion (dorsal columns and corticospinal tracts initially spared)

---

## 🔑 Decussation Anatomy — Why Contralateral?

```
Pain stimulus on RIGHT side of body:
    Right Aδ/C fibre → Right DRG → Right dorsal horn (laminae I, II, V)
    ↓
Crosses via anterior white commissure (1-2 segments ABOVE entry)
    ↓
Ascends in LEFT anterolateral column → LEFT VPL thalamus → LEFT S1 cortex
→ Patient perceives pain on RIGHT side (contralateral cortex processes)
```

The 1-2 segment oblique course of the crossing fibres means:
- A spinothalamic lesion produces pain/temperature loss **contralateral** and **1-2 levels below** the lesion

## 🔑 Comparison: Spinothalamic vs Dorsal Column

| Feature | Spinothalamic Tract | Dorsal Column–Medial Lemniscal |
|---------|--------------------|---------------------------------|
| Modalities | Pain, temperature, crude touch, itch | Fine touch, vibration, proprioception, 2-point discrimination |
| Primary afferents | Aδ, C fibres | Aβ fibres (large, myelinated) |
| 1st synapse | Dorsal horn (ipsilateral) | Nucleus gracilis/cuneatus (medulla) |
| Decussation | Spinal cord (anterior white commissure, 1-2 levels above entry) | Medulla (internal arcuate fibres) |
| Ascending column | Contralateral anterolateral | Ipsilateral dorsal column |
| Lesion produces | Contralateral loss below | Ipsilateral loss below |
| Speed | Slower (unmyelinated C) | Faster (large myelinated Aβ) |

## 🏥 Clinical Significance

| Condition | Mechanism | Findings |
|-----------|-----------|---------|
| **Brown-Séquard syndrome** (cord hemisection) | Contralateral spinothalamic loss + ipsilateral dorsal column/UMN loss | Contralateral pain/temp loss; ipsilateral proprioception/fine touch loss + UMN signs |
| **Syringomyelia** | Central cord cavity → bilateral crossing fibre disruption at that level | Bilateral suspended (cape) pain/temperature loss; dorsal columns spared early; later: UMN + LMN |
| **Anterior cord syndrome** | Anterior spinal artery occlusion → bilateral spinothalamic + corticospinal loss | Bilateral pain/temp loss + motor loss; dorsal columns (proprioception/vibration) PRESERVED |
| **Complete cord transection** | All ascending/descending tracts | Loss of all modalities below lesion + autonomic dysfunction |
| **Lateral medullary (Wallenberg) syndrome** | PICA/vertebral artery; spinothalamic + spinal trigeminal in lateral medulla | Ipsilateral face pain/temp loss (CN V nucleus) + **contralateral body** pain/temp loss (STT) = "crossed" sensory loss |
| **Thalamic (VPL) lesion** | Stroke in VPL (PCA territory) | Contralateral hemibody loss of all sensory modalities + Dejerine-Roussy thalamic pain |
| **Surgical cordotomy** | Deliberate anterolateral incision | Contralateral pain/temp loss for cancer pain relief (below and contralateral to level) |
