# 🧬 Neurodegenerative Diseases

> **Progressive, irreversible neuronal loss defined by protein misfolding and aggregation — each disease targets a specific protein, pathway, and brain region, producing characteristic clinical syndromes.**

## 🔗 Key Connections
- [Basal Ganglia](./anatomyLink/Nervous System/Central Nervous System/Basal Ganglia.md) — nigrostriatal degeneration in Parkinson's; striatal loss in Huntington's
- [Hippocampus](./anatomyLink/Nervous System/Neuro System (Lippincott)/04 - Brain Regions/Hippocampus.md) — earliest target in Alzheimer's disease
- [Cerebral Cortex](./anatomyLink/Nervous System/Neuro System (Lippincott)/04 - Brain Regions/Cerebral Cortex.md) — widespread cortical atrophy in AD, FTD, ALS-FTD
- [Motor Pathway](./anatomyLink/Nervous System/Pathways/Motor Pathway.md) — upper + lower motor neuron degeneration in ALS
- [Neurotransmitters](./anatomyLink/Nervous System/Neuro System (Lippincott)/01 - Cellular Neuroscience/Neurotransmitters.md) — dopamine (PD), acetylcholine (AD), glutamate (excitotoxicity)
- [Glial Cells](./anatomyLink/Nervous System/Neuro System (Lippincott)/01 - Cellular Neuroscience/Glial Cells.md) — demyelination in MS; reactive gliosis in all
- [Synaptic Transmission](./anatomyLink/Nervous System/Neuro System (Lippincott)/01 - Cellular Neuroscience/Synaptic Transmission.md) — synaptic loss = best correlate of cognitive decline in AD
- [Sleep and Wakefulness](./anatomyLink/Nervous System/Neuro System (Lippincott)/05 - Higher Functions/Sleep and Wakefulness.md) — REM sleep behaviour disorder as prodromal marker for PD/DLB
- [Amygdala](./anatomyLink/Nervous System/Neuro System (Lippincott)/04 - Brain Regions/Amygdala.md) — Lewy bodies in DLB; tau in AD
- [Epilepsy](./anatomyLink/Nervous System/Neuro System (Lippincott)/06 - Clinical Neurology/Epilepsy.md) — seizures occur in late AD and some genetic forms
- [Stroke Syndromes](./anatomyLink/Nervous System/Neuro System (Lippincott)/06 - Clinical Neurology/Stroke Syndromes.md) — vascular dementia differential; multi-infarct

## 🗂️ Key Information

| Property | Detail |
|----------|--------|
| Definition | Progressive neuronal loss; protein aggregation (proteinopathy) |
| Common feature | No disease-modifying treatment for most (2025) |
| Classification | By culprit protein: amyloid/tau (AD), α-synuclein (PD/DLB/MSA), TDP-43 (ALS/FTD), huntingtin (HD), prion (CJD) |
| Shared mechanism | Misfolded protein seeding, propagation (prion-like), synaptic failure, mitochondrial dysfunction |
| Leading cause of dementia | Alzheimer's (60-70%) > Vascular > DLB > FTD |

## 🔑 Alzheimer's Disease (AD)

### Epidemiology
- Most common dementia (60-70%); 2nd most common neurodegenerative disease overall
- Prevalence doubles every ~5 years after age 65; affects ~50 million worldwide

### Pathology
| Feature | Detail |
|---------|--------|
| Amyloid plaques | Aβ42 from APP cleavage (BACE1 → γ-secretase/presenilin); Aβ oligomers most toxic; plaques = amyloid core + dystrophic neurites + reactive glia |
| Neurofibrillary tangles (NFTs) | Hyperphosphorylated tau → microtubule destabilization → intraneuronal filaments |
| Braak staging | I-II: entorhinal/hippocampal; III-IV: limbic; V-VI: neocortical |
| Synaptic loss | Best correlate of cognitive decline (more than plaque/tangle burden) |
| Cholinergic deficit | Nucleus basalis of Meynert (Ch4) → cortex/hippocampus; basis for ChEI therapy |
| Neuroinflammation | Microglial activation, complement, TREM2 variants (AD risk gene — microglial Aβ clearance) |

### Amyloid Cascade Hypothesis
```
APP → BACE1 cleavage → Aβ42 overproduction OR ↓clearance
    ↓
Aβ oligomers (most toxic) → synaptic dysfunction
    ↓
Amyloid plaques → neuroinflammation
    ↓
Tau hyperphosphorylation → NFT spread (Braak staging)
    ↓
Neuronal/synaptic loss → cognitive decline → dementia
```

### Genetics
| Gene/Factor | Chromosome | Effect |
|-------------|-----------|--------|
| APOE ε4 | 19q13 | Most common genetic risk; ↑Aβ aggregation, ↓clearance; ε4/ε4 ~10× risk; ε2 protective |
| APP mutation | 21q21 | Familial early-onset; trisomy 21 (Down syndrome) → AD by 40s from extra APP copy |
| PSEN1 | 14q24 | Most common familial mutation; early-onset (~45y); encodes presenilin-1 (γ-secretase) |
| PSEN2 | 1q42 | Rare familial; later onset than PSEN1 |

### Clinical Progression
| Stage | Features |
|-------|---------|
| Preclinical | Aβ accumulation (PET+) years before symptoms; no cognitive deficit |
| MCI (amnestic) | Episodic memory impairment > expected for age; hippocampal atrophy on MRI |
| Mild AD | Memory loss + anomia; can still manage ADLs; insight present |
| Moderate AD | Language, visuospatial, executive deficits; gets lost; dressing apraxia; loses insight |
| Severe AD | Non-ambulatory, incontinence, dysphagia, myoclonus; death from aspiration pneumonia |

### Diagnosis
- MRI: hippocampal + entorhinal atrophy; posterior parietal/temporal thinning
- FDG-PET: hypometabolism in posterior cingulate, precuneus, parietal, temporal cortex
- Amyloid PET (florbetapir/florbetaben): positive amyloid burden
- CSF biomarkers: ↓Aβ42 (sequestered in plaques), ↑total tau, ↑phospho-tau (p-tau181)
- Blood biomarkers (emerging): plasma p-tau217, Aβ42/40 ratio, GFAP, NfL

### Treatment
| Drug | Class | Mechanism | Use |
|------|-------|-----------|-----|
| Donepezil | ChEI | Inhibits AChE → ↑synaptic ACh | Mild-severe AD |
| Rivastigmine | ChEI | Inhibits AChE + BuChE | Mild-moderate; patch for PDD |
| Galantamine | ChEI | AChE inhibition + nicotinic allosteric modulation | Mild-moderate |
| Memantine | NMDA antagonist | Blocks tonic NMDA activation → ↓excitotoxicity | Moderate-severe |
| Lecanemab | Anti-Aβ mAb | Clears Aβ protofibrils; ~27% slowing of decline | Early AD (FDA 2023); risk: ARIA |

## 🔑 Parkinson's Disease (PD)

### Epidemiology & Pathology
- 2nd most common neurodegenerative disease; 1-2% prevalence >65 years
- **Lewy bodies**: cytoplasmic inclusions of α-synuclein + ubiquitin; dopaminergic neurons of substantia nigra pars compacta (SNc) most affected
- Loss of nigrostriatal DA → imbalance of direct (D1/"Go") vs indirect (D2/"Stop") pathways → ↑STN → ↑GPi/SNr → ↑thalamic inhibition → ↓motor cortex activity

### Braak Staging (Lewy body propagation — prion-like spread)
| Stage | Regions | Symptoms |
|-------|---------|---------|
| 1-2 | Dorsal motor nucleus CN X; olfactory bulb | Constipation, anosmia, RBD (prodromal) |
| 3-4 | Substantia nigra; locus coeruleus; amygdala | Motor symptoms (TRAP), depression |
| 5-6 | Neocortex | Dementia, hallucinations (PDD) |

### Motor Features — TRAP
| Sign | Description |
|------|-------------|
| **T**remor | Rest tremor, 4-6 Hz, "pill-rolling," asymmetric onset, disappears with action |
| **R**igidity | Cogwheel (tremor on lead-pipe), throughout ROM, "gegenhalten" |
| **A**kinesia/Bradykinesia | Slowness, reduced amplitude — micrographia, hypomimia, hypophonia, festinating gait |
| **P**ostural instability | Late feature; pull test positive; retropulsion; falls |

### Non-Motor Features
- **Prodromal** (precede motor by years): anosmia, constipation, RBD (acts out dreams — strongest predictor), depression, orthostatic hypotension
- **Later**: cognitive impairment → PD dementia (PDD, >1 year after motor onset; if dementia ≤1 year = DLB), hallucinations (visual), psychosis, autonomic (OH, urinary urgency, sweating), pain, fatigue

### Treatment
| Drug/Therapy | Mechanism | Notes |
|-------------|-----------|-------|
| L-DOPA + carbidopa | DA precursor; carbidopa = peripheral DDC inhibitor | Gold standard; long-term: wearing-off, dyskinesias, on-off |
| Dopamine agonists (ropinirole, pramipexole) | D2/D3 agonist | First-line younger patients; impulse control disorder risk |
| MAO-B inhibitors (rasagiline, selegiline) | ↓DA catabolism | Mild benefit; possible neuroprotection (unproven) |
| COMT inhibitors (entacapone) | ↓L-DOPA peripheral breakdown | Extends L-DOPA effect; wearing-off |
| Amantadine | NMDA antagonist + DA release | Reduces peak-dose dyskinesias |
| DBS (STN or GPi) | High-frequency stimulation (~130 Hz) functionally inhibits target | Controls tremor/rigidity/bradykinesia; not dementia or autonomic |

### Parkinson-Plus Syndromes (atypical parkinsonism — poor L-DOPA response)
| Syndrome | Protein | Key Features |
|---------|---------|-------------|
| PSP (Progressive Supranuclear Palsy) | Tau (4R) | Vertical gaze palsy (downward first), falls backward, Richardson's syndrome, "hummingbird sign" MRI |
| MSA (Multiple System Atrophy) | α-synuclein | Autonomic failure + parkinsonism/cerebellar; "hot cross bun" MRI pons |
| DLB (Dementia with Lewy Bodies) | α-synuclein | Dementia ≤1y of parkinsonism; fluctuating cognition; visual hallucinations; RBD; neuroleptic sensitivity |
| CBS/CBD (Corticobasal Syndrome) | Tau (4R) | Asymmetric, alien limb, apraxia, cortical sensory loss |

## 🔑 Huntington's Disease (HD)

| Property | Detail |
|----------|--------|
| Inheritance | Autosomal dominant; full penetrance |
| Gene | HTT (huntingtin), chromosome 4p16.3 |
| Mutation | CAG trinucleotide repeat expansion: normal ≤26, HD ≥36; anticipation (especially paternal) |
| Onset | Inversely proportional to CAG repeat length; mean ~40y; juvenile HD >60 repeats |
| Pathology | Mutant huntingtin (mHTT) toxic gain-of-function; striatal medium spiny neuron loss (indirect pathway MSNs first → chorea) |
| MRI | Caudate atrophy → "boxcar/bat-wing" ventricles |

### Clinical Triad
| Domain | Features |
|--------|---------|
| Motor | Chorea (early — involuntary irregular flowing movements); later: rigidity, dystonia, bradykinesia; oculomotor (saccade initiation delay); dysarthria; dysphagia |
| Psychiatric | Depression (↑suicide 4×), irritability/aggression, apathy, anxiety, OCD-like, psychosis — often precede motor symptoms |
| Cognitive | Subcortical dementia: slowed processing, executive dysfunction, memory retrieval deficit, visuospatial decline; insight preserved until late |

### Treatment
- No disease-modifying treatment
- **Tetrabenazine / Deutetrabenazine**: VMAT2 inhibitor → ↓presynaptic DA → reduces chorea; risk: depression/suicidality; contraindicated with MAO inhibitors
- SSRIs/SNRIs for depression; antipsychotics for psychiatric; MDT care

## 🔑 ALS (Amyotrophic Lateral Sclerosis)

| Property | Detail |
|----------|--------|
| Pathology | Combined UMN + LMN degeneration; TDP-43 cytoplasmic inclusions (97%); SOD1 mutations (familial 20%); C9orf72 expansion (most common familial + sporadic genetic cause; also FTD) |
| Spared systems | Sensory, bowel/bladder, eye movements (until terminal), cognition (usually) |
| Prognosis | Death from respiratory failure; mean 2-5 years from onset |

### Clinical Signs
| Feature | UMN | LMN |
|---------|-----|-----|
| Tone | Spasticity | Flaccidity |
| Reflexes | Hyperreflexia | Hyporeflexia/areflexia |
| Babinski | Positive | Absent |
| Fasciculations | Absent | Present |
| Atrophy | Mild | Severe |
| Onset pattern | Bulbar (worse prognosis) or spinal (limb) | |

### Treatment
| Drug | Mechanism | Benefit |
|------|-----------|---------|
| Riluzole | ↓Presynaptic glutamate release (Na+ channel) | +2-3 months survival |
| Edaravone | Free radical scavenger | Slows functional decline in selected patients |
| Tofersen | Antisense oligonucleotide (SOD1-ALS) | ↓SOD1 protein + NfL (biomarker) |
| Ventilatory support | NIV (BiPAP) | Extends life significantly |

## 🔑 Multiple Sclerosis (MS)

| Property | Detail |
|----------|--------|
| Mechanism | Autoimmune T-cell (Th1/Th17) + B-cell attack on oligodendrocytes/myelin |
| Lesions | Periventricular (Dawson fingers — perpendicular to ventricles on sagittal FLAIR), juxtacortical, infratentorial, spinal cord |
| CSF | Oligoclonal bands (IgG) in 85-95%; ↑IgG index |

### Clinical Hallmarks
| Sign | Lesion Site |
|------|------------|
| Optic neuritis | CN II / optic nerve — painful monocular vision loss, RAPD, Uhthoff's |
| INO (internuclear ophthalmoplegia) | MLF — impaired ipsilateral adduction + contralateral nystagmus |
| Lhermitte's sign | Cervical cord — neck flexion → electric shocks down spine |
| Uhthoff's phenomenon | Any — symptoms worsen with heat (↑conduction block in demyelinated axons) |
| Spasticity, bladder dysfunction, fatigue | Spinal cord / widespread |

### Disease-Modifying Therapies
| Drug | Mechanism | Efficacy | Key Risk |
|------|-----------|---------|---------|
| Interferon-β | Anti-inflammatory, ↓MMP, BBB stabilisation | Moderate | Flu-like symptoms, injection site |
| Glatiramer acetate | MBP mimic → regulatory T cells | Moderate | Injection site |
| Natalizumab | Anti-α4 integrin → ↓lymphocyte CNS entry | High | PML (JC virus — test JC antibody index) |
| Ocrelizumab | Anti-CD20 B-cell depletion | High (also PPMS) | Infusion reactions, infections |
| Siponimod/Ozanimod | S1P receptor — traps lymphocytes in lymph nodes | High | Bradycardia, macular oedema |

| Disease | Protein | Region Lost | Key Feature | Genetic Cause |

| Alzheimer's | Aβ + tau | Hippocampus → cortex | Episodic memory loss first | APOE ε4 (risk); PSEN1 (familial) |
| Parkinson's | α-synuclein | SNc nigrostriatal | Rest tremor + bradykinesia | LRRK2, PINK1, Parkin, SNCA |

| ALS | TDP-43 / SOD1 | UMN + LMN | Mixed UMN+LMN, no sensory | C9orf72 > SOD1 (familial) |
| FTD | TDP-43 or tau | Frontal + temporal | Behavioural change / aphasia | C9orf72, MAPT, GRN |
| MS | Myelin (autoimmune) | White matter (CNS) | Relapses + remissions; INO | HLA-DRB1*15:01 (risk) |
| PSP | Tau (4R) | Brainstem + BG | Vertical gaze palsy, falls back | Sporadic (MAPT H1 haplotype) |
| CJD | Prion (PrPSc) | Widespread cortex | Rapidly progressive dementia + myoclonus | PRNP (familial), iatrogenic, sporadic |
