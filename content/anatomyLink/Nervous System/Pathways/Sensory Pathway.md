# Sensory Pathway

> There are two major ascending sensory pathways: the **dorsal column-medial lemniscal pathway** (fine touch, vibration, proprioception) and the **spinothalamic tract** (pain, temperature, crude touch). They decussate at different levels — critical for lesion localization.

## 🔗 Key Connections
- [Spinal Cord](./anatomyLink/Nervous System/Central Nervous System/Spinal Cord.md) — both pathways ascend within the cord
- [Brainstem](./anatomyLink/Nervous System/Central Nervous System/Brainstem.md) — dorsal columns decussate in medulla; spinothalamic already crossed in cord
- [Brain](./anatomyLink/Nervous System/Central Nervous System/Brain.md) — both reach the thalamus (VPL nucleus) → primary somatosensory cortex (postcentral gyrus)

## 🗂️ Two Major Pathways

### 1. Dorsal Column — Medial Lemniscal Pathway
**Modalities**: Fine touch, vibration, proprioception, 2-point discrimination

```
1st neuron: Sensory receptor → DRG (cell body) → Enters cord → Ascends IPSILATERAL dorsal column
    Fasciculus gracilis (legs, T6 and below) + Fasciculus cuneatus (arms, above T6)
    ↓ Medulla
2nd neuron: Nucleus gracilis/cuneatus (medulla) → DECUSSATES as internal arcuate fibers
    → Medial lemniscus (ascends contralaterally through brainstem)
    ↓ Thalamus
3rd neuron: VPL nucleus of thalamus → Posterior limb internal capsule
    → Primary somatosensory cortex (postcentral gyrus, S1)
```

**Decussation**: Medulla (nucleus gracilis/cuneatus)

### 2. Spinothalamic Tract (Anterolateral System)
**Modalities**: Pain, temperature, crude touch, pressure

```
1st neuron: Sensory receptor → DRG (cell body) → Enters cord → Ascends 1–2 levels in Lissauer's tract
    ↓ Posterior horn (substantia gelatinosa — Rexed laminae I, II, V)
2nd neuron: DECUSSATES within 1–2 spinal cord levels via anterior commissure
    → Contralateral lateral (pain/temp) + anterior (crude touch) funiculus
    → Ascends as spinothalamic tract through cord, brainstem
    ↓ Thalamus
3rd neuron: VPL nucleus → Posterior limb internal capsule → S1 cortex
```

**Decussation**: Spinal cord (1–2 levels above entry)

## 🔑 Comparison Table
| Feature | Dorsal Columns | Spinothalamic |
|---------|---------------|---------------|
| Modalities | Fine touch, vibration, proprioception | Pain, temperature, crude touch |
| 1st neuron | DRG → ipsilateral dorsal column | DRG → posterior horn |
| Decussation | Medulla | Spinal cord (1–2 levels above) |
| 2nd neuron | Nucleus gracilis/cuneatus (medulla) | Posterior horn cells |
| Ascends as | Medial lemniscus | Spinothalamic tract |
| 3rd neuron | VPL thalamus → S1 cortex | VPL thalamus → S1 cortex |

## 🔑 Somatotopy in Spinal Cord
**Dorsal columns**: Sacral fibers most medial (near midline); cervical most lateral
**Spinothalamic**: Sacral fibers most lateral; cervical most medial
- Explains "sacral sparing" in central cord syndrome (sacral fibers at periphery)

## 🏥 Clinical Significance — Lesion Localization

| Lesion | Dorsal Column Loss | Spinothalamic Loss | Pattern |
|--------|-------------------|-------------------|---------|
| **Spinal cord hemisection (Brown-Séquard)** | Ipsilateral (same side, below) | Contralateral (opposite side, 1–2 levels below) | Dissociated sensory loss |
| **Central cord syndrome** | Preserved initially | Bilateral loss at level (cape distribution) | Syrinx, central cord injury |
| **Posterior cord syndrome** | Bilateral loss below | Preserved | B12 deficiency, syphilis (tabes dorsalis) |
| **Anterior cord syndrome** | Preserved | Bilateral loss below | Anterior spinal artery occlusion |
| **Thalamic lesion** | Contralateral hemibody | Contralateral hemibody | Both pathways → thalamic pain syndrome |
| **Cortical lesion** | Contralateral (discriminative touch) | Contralateral (crude) | Both; cortex better for discrimination |

## 🔑 Special Sensory Tests
| Test | Pathway Tested |
|------|---------------|
| Vibrating tuning fork | Dorsal columns |
| Romberg test | Dorsal columns (balance with eyes closed) |
| Pinprick | Spinothalamic |
| Temperature | Spinothalamic |
| 2-point discrimination | Dorsal columns + cortex |
| Proprioception (joint position) | Dorsal columns |
