# Basal Ganglia

> The **basal ganglia** are a group of subcortical nuclei deep within the [Brain](./anatomyLink/Nervous System/Central Nervous System/Brain.md) that regulate **voluntary movement initiation, motor learning, and habit formation**. They work via inhibitory loops — dysfunction causes either too much movement (hyperkinetic) or too little (hypokinetic).

## 🔗 Key Connections
- [Brain](./anatomyLink/Nervous System/Central Nervous System/Brain.md) — embedded in cerebral hemispheres and diencephalon
- [Brainstem](./anatomyLink/Nervous System/Central Nervous System/Brainstem.md) — substantia nigra is functionally part of the basal ganglia circuit
- Motor cortex → striatum → output nuclei → thalamus → motor cortex (loop)

## 🗂️ Components

### Nuclei
| Nucleus | Location | Notes |
|---------|----------|-------|
| **Caudate nucleus** | C-shaped; lateral to lateral ventricle | Cognition, reward |
| **Putamen** | Lateral to internal capsule | Movement control (main input from motor cortex) |
| **Caudate + Putamen** = **Striatum** | — | Main INPUT structure |
| **Globus Pallidus Internus (GPi)** | Medial | Main OUTPUT to thalamus (inhibitory) |
| **Globus Pallidus Externus (GPe)** | Lateral | Part of indirect pathway |
| **Caudate + Putamen + GP** = **Lentiform nucleus** | — | |
| **Subthalamic nucleus (STN)** | Diencephalon | Indirect pathway; DBS target for Parkinson's |
| **Substantia Nigra Pars Compacta (SNc)** | Midbrain | Dopamine production → striatum |
| **Substantia Nigra Pars Reticulata (SNr)** | Midbrain | Output (like GPi) |

### Circuits

#### Direct Pathway (Facilitates Movement — "GO")
```
Cortex → Striatum →(−) GPi/SNr →(−) Thalamus → Cortex ↑
         (D1 receptors; dopamine EXCITES)
```
Net effect: **Thalamus disinhibited → Movement facilitated**

#### Indirect Pathway (Inhibits Movement — "STOP")
```
Cortex → Striatum →(−) GPe →(−) STN →(+) GPi/SNr →(−) Thalamus → Cortex ↓
         (D2 receptors; dopamine INHIBITS)
```
Net effect: **Thalamus inhibited → Movement suppressed**

#### Dopamine Balance
- **Dopamine** from SNc activates D1 (direct — GO) and inhibits D2 (indirect — STOP)
- Net dopamine effect: **Facilitates movement**
- Loss of dopamine (Parkinson's) → reduced direct, increased indirect → **bradykinesia**

## 🔑 Neurotransmitters
| Pathway | Transmitter | Effect |
|---------|-------------|--------|
| Striatum → GPi/SNr | GABA | Inhibitory |
| GPi/SNr → Thalamus | GABA | Inhibitory |
| SNc → Striatum | Dopamine | D1 excite / D2 inhibit |
| STN → GPi | Glutamate | Excitatory |
| Thalamus → Cortex | Glutamate | Excitatory |

## 🏥 Clinical Significance

### Hypokinetic Disorders (Too Little Movement)
| Disease | Lesion | Mechanism | Features |
|---------|--------|-----------|----------|
| **Parkinson's disease** | SNc degeneration | ↓Dopamine → ↑GPi inhibition of thalamus | TRAP: Tremor (rest, 4–6 Hz), Rigidity (cogwheel), Akinesia/bradykinesia, Postural instability |
| **Drug-induced parkinsonism** | D2 blockade (antipsychotics) | Blocks indirect pathway inhibition | Same features as PD; reversible |

### Hyperkinetic Disorders (Too Much Movement)
| Disease | Lesion | Mechanism | Features |
|---------|--------|-----------|----------|
| **Huntington's disease** | Striatum (caudate first) | ↓Indirect pathway → thalamus disinhibited | Chorea (random, dance-like), dementia, psychiatric; CAG repeat on chr 4 |
| **Hemiballismus** | Contralateral STN | ↓STN excitation of GPi → thalamus disinhibited | Violent flinging movements of proximal limb; most common cause: lacunar stroke in STN |
| **Athetosis** | Striatum/putamen | Variable | Slow, writhing distal movements |
| **Wilson's disease** | Putamen (copper deposition) | Copper toxicity | Tremor + liver disease + Kayser-Fleischer rings |

### Treatments
- **Parkinson's**: L-DOPA + carbidopa; dopamine agonists; MAO-B inhibitors; DBS of STN or GPi
- **Huntington's**: Tetrabenazine (depletes monoamines); no disease-modifying treatment
- **Hemiballismus**: Haloperidol; usually resolves spontaneously
