# Limbic System

> The **limbic system** is a set of interconnected brain structures forming the emotional and memory center of the [[Brain]]. Key structures include the **hypothalamus, hippocampus, amygdala, cingulate gyrus**, and **fornix**.

## 🔗 Key Connections
- [[Brain]] — medial temporal lobe + diencephalon
- [[Brainstem]] — hypothalamus controls autonomic output via brainstem
- [[Cranial Nerves]] — CN I (olfactory) has direct limbic projections (emotion + smell)

## 🗂️ Key Structures

### Hypothalamus
> The hypothalamus is the **master regulator** of the autonomic nervous system and endocrine system, maintaining homeostasis.

| Nucleus | Function |
|---------|----------|
| **Anterior** | Cooling, parasympathetic; ADH + oxytocin synthesis |
| **Posterior** | Warming, sympathetic activation |
| **Lateral** | Hunger (lesion → anorexia) |
| **Ventromedial** | Satiety center (lesion → hyperphagia/obesity) |
| **Suprachiasmatic** | Circadian rhythm; receives direct retinal input |
| **Supraoptic + Paraventricular** | ADH + oxytocin synthesis → posterior pituitary |
| **Mammillary bodies** | Memory circuit (Papez); damaged in Wernicke's encephalopathy |

**Mnemonic (TAN HATS):** Thirst/ADH, Adenohypophysis (anterior pituitary), Neurohypophysis (posterior pituitary), Hunger, Autonomic, Temperature, Sexual behavior

#### Hypothalamic-Pituitary Axis
```
Hypothalamus → Releasing hormones → Anterior pituitary → Trophic hormones → Target glands
Hypothalamus → ADH + Oxytocin → Posterior pituitary → Direct release
```

### Hippocampus
| Property | Detail |
|----------|--------|
| Location | Medial temporal lobe (floor of inferior horn of lateral ventricle) |
| Function | **Explicit (declarative) memory** formation; spatial navigation |
| Circuit | Papez circuit: Hippocampus → Fornix → Mammillary bodies → Thalamus → Cingulate → Hippocampus |
| Vulnerability | Most sensitive to hypoxia/ischemia ("CA1 sector" most vulnerable) |

### Amygdala
| Property | Detail |
|----------|--------|
| Location | Anterior temporal lobe, medial |
| Function | **Fear conditioning**, emotional memory, threat detection |
| Input | All sensory modalities + hippocampus |
| Output | Hypothalamus (autonomic fear response), brainstem, prefrontal cortex |
| Bilateral lesion | **Klüver-Bucy syndrome**: placidity, hypersexuality, hyperorality, visual agnosia |

### Cingulate Gyrus
| Division | Function |
|----------|----------|
| **Anterior cingulate (ACC)** | Emotion regulation, conflict monitoring, pain affect |
| **Posterior cingulate** | Spatial orientation, episodic memory retrieval |

### Fornix
- C-shaped white matter tract connecting hippocampus → mammillary bodies
- Damage → anterograde amnesia (can't form new memories)

## 🔑 Papez Circuit (Memory + Emotion Loop)
```
Hippocampus → Fornix → Mammillary bodies → Thalamus (anterior) → Cingulate gyrus → Parahippocampal cortex → Hippocampus
```

## 🔑 Olfaction and the Limbic System
- Olfaction (CN I) is the ONLY sense that bypasses the thalamus and projects **directly** to the limbic system (amygdala, piriform cortex)
- Explains why smell powerfully triggers emotional memories

## 🏥 Clinical Significance

| Condition | Structure | Features |
|-----------|-----------|----------|
| **Alzheimer's disease** | Hippocampus (first) → cortex | Anterograde amnesia → global dementia; amyloid plaques + tau tangles |
| **Wernicke's encephalopathy** | Mammillary bodies + thalamus (thiamine deficiency) | Confusion, ophthalmoplegia, ataxia (triad) |
| **Korsakoff syndrome** | Mammillary bodies (chronic) | Anterograde + retrograde amnesia; confabulation |
| **Bilateral hippocampal damage** | Hippocampus (e.g., H.M. patient) | Inability to form new declarative memories; remote memory preserved |
| **PTSD** | Amygdala hyperactivity + hippocampal atrophy | Fear dysregulation, intrusive memories |
| **Temporal lobe epilepsy** | Hippocampus/amygdala | Complex partial seizures; aura (déjà vu, fear, smell); automatisms |
| **Klüver-Bucy syndrome** | Bilateral amygdala | Placidity, hypersexuality, hyperorality, visual agnosia |
| **Diabetes insipidus** | Posterior hypothalamus / posterior pituitary | ↓ADH → massive dilute urine |
| **SIADH** | Hypothalamus | ↑ADH → hyponatremia |
