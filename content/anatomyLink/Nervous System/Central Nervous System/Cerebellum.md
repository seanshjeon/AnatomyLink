# Cerebellum

> The **cerebellum** ("little brain") occupies the posterior fossa, comprising ~10% of brain volume but containing ~50% of all neurons. It coordinates **movement, balance, and motor learning** — all damage causes **ipsilateral** deficits (unlike cerebral lesions).

## 🔗 Key Connections
- [Brain](./anatomyLink/Nervous-System/Central-Nervous-System/Brain.md) — connected superiorly via tentorium cerebelli
- [Brainstem](./anatomyLink/Nervous-System/Central-Nervous-System/Brainstem.md) — connected via three cerebellar peduncles
- [Spinal Cord](./anatomyLink/Nervous-System/Central-Nervous-System/Spinal-Cord.md) — receives proprioceptive input via spinocerebellar tracts
- [Cranial Nerves](./anatomyLink/Nervous-System/Peripheral-Nervous-System/Cranial-Nerves.md) — CN VIII (vestibular input to flocculonodular lobe)

## 🗂️ Anatomy

### Lobes
| Lobe | Location | Function |
|------|----------|----------|
| **Anterior lobe** | Superior, anterior | Spinocerebellum — regulates muscle tone + limb movement |
| **Posterior lobe** | Largest; inferior | Cerebrocerebellum — fine motor skill, motor planning |
| **Flocculonodular lobe** | Inferior; most primitive | Vestibulocerebellum — balance + eye movement |

### Functional Divisions (Clinically Important)
| Division | Input | Output | Function |
|----------|-------|--------|----------|
| **Vestibulocerebellum** (flocculonodular) | Vestibular system (CN VIII) | Vestibular nuclei | Balance, eye movements |
| **Spinocerebellum** (vermis + paravermis) | Spinal cord (spinocerebellar) | Red nucleus, reticular formation | Limb + trunk coordination; muscle tone |
| **Cerebrocerebellum** (lateral hemispheres) | Motor cortex via pons | Thalamus → motor cortex | Motor planning, timing, skill learning |

### Deep Cerebellar Nuclei (Output)
From lateral → medial: **Dentate → Emboliform → Globose → Fastigial**
Mnemonic: **"Don't Eat Greasy Food"**
| Nucleus | Location | Pathway |
|---------|----------|---------|
| **Dentate** | Lateral | → Thalamus (VL) → motor cortex (via SCP) |
| **Emboliform + Globose** (= Interposed) | Middle | → Red nucleus → rubrospinal tract |
| **Fastigial** | Medial (vermis) | → Vestibular nuclei + reticular formation |

### Cortical Layers
1. **Molecular layer** — sparse cells; Purkinje dendrites + parallel fibers
2. **Purkinje cell layer** — single layer of large Purkinje cells (only OUTPUT of cortex — inhibitory GABAergic → deep nuclei)
3. **Granule cell layer** — densest neuron packing in the brain; granule cells receive mossy fiber input

### Inputs
- **Mossy fibers** — From spinal cord + pons (most inputs); synapse on granule cells
- **Climbing fibers** — From inferior olive ONLY; powerful synapse on Purkinje cells; motor error learning

## 🔑 Cerebellar Circuits
```
Motor cortex → Pons → Mossy fibers → Granule cells → Parallel fibers → Purkinje cells
                                                                              ↓ (inhibit)
Inferior olive → Climbing fibers ─────────────────────────────────> Deep cerebellar nuclei
                                                                              ↓
                                                              Thalamus → Motor cortex (feedback)
```

## 🏥 Clinical Significance

### Signs of Cerebellar Damage (Always Ipsilateral)
| Sign | Description |
|------|-------------|
| **Ataxia** | Uncoordinated gait; wide-based, staggering |
| **Dysmetria** | Over/undershooting targets (finger-nose test) |
| **Intention tremor** | Tremor that worsens as limb approaches target |
| **Dysdiadochokinesia** | Inability to perform rapid alternating movements |
| **Dysarthria** | Scanning/staccato speech (cerebellar dysarthria) |
| **Nystagmus** | Fast phase toward lesion side |
| **Hypotonia** | Decreased muscle tone (ipsilateral) |
| **Truncal ataxia** | Vermis lesion → can't sit/stand without falling |

### Localization
| Location | Signs |
|----------|-------|
| **Vermis** | Truncal ataxia, gait ataxia (midline); seen in alcoholic cerebellar degeneration |
| **Hemispheres** | Ipsilateral limb ataxia, dysmetria, intention tremor |
| **Flocculonodular** | Nystagmus, vertigo, gait imbalance (like vestibular disease) |

### Causes of Cerebellar Ataxia
- **Acute**: stroke (PICA/AICA/SCA), MS, toxins (phenytoin, alcohol intoxication)
- **Chronic**: alcoholic cerebellar degeneration (anterior vermis), spinocerebellar ataxias (SCAs), paraneoplastic
- **Pediatric**: medulloblastoma (vermis → truncal ataxia), pilocytic astrocytoma (hemisphere)
