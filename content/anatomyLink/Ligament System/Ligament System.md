# 🦷 Ligament System

> **Ligaments** are dense connective tissue bands that connect bone to bone, providing joint stability and guiding motion. Unlike tendons (muscle→bone), ligaments are purely structural stabilizers.

| Ligament                            | Function                                 |
| ----------------------------------- | ---------------------------------------- |

| [Ligamentum Flavum](./anatomyLink/Ligament System/Spine/Ligamentum Flavum.md)               | Connects laminae; maintains posture      |
| [Ligamentum Nuchae](./anatomyLink/Ligament System/Spine/Ligamentum Nuchae.md)               | Cervical spine support                   |
