# Kidney

> The kidneys are **paired retroperitoneal organs** (~11 × 6 × 3 cm; ~150 g each) lying at the T12–L3 level, flanking the vertebral column. They are the primary organs of **urine formation**, regulating fluid balance, electrolytes, acid-base status, and blood pressure.

## 🔗 Key Connections
- [[00 - AnatomyLink Home]] → Urinary System
- Medial structure: Renal hilum → renal pelvis → ureter → urinary bladder
- Blood supply: **Renal arteries** (from abdominal aorta at L1–L2; right is longer, crossing IVC)
- Venous drainage: Renal veins → IVC (left renal vein crosses aorta anteriorly, receives left gonadal + left adrenal vein)
- Lymphatics: Para-aortic (lateral aortic) nodes
- Relations:
  - Right kidney: liver (hepatorenal space / Morison's pouch), duodenum (medially), ascending colon (inferiorly)
  - Left kidney: spleen, stomach, pancreas (tail), descending colon
- Superior: Adrenal (suprarenal) glands
- Skeletal relations: [[Lumbar Vertebrae]] (L1–L3), 12th [[Ribs]] (posteriorly), [[Vertebral Column]]
- Peritoneal relations: Retroperitoneal (behind the peritoneum)

## 🔬 Structure

### External
| Layer (outside→in) | Description |
|--------------------|-------------|
| Pararenal fat | Outermost; can be minimal |
| **Gerota's fascia** (renal fascia) | Fibrous envelope enclosing kidney + adrenal |
| Perirenal fat | Fatty cushion around kidney |
| **Renal capsule** | Fibrous capsule directly on kidney surface |

### Internal (Coronal Section)

| Region | Description |
|--------|-------------|
| **Cortex** | Outer; contains glomeruli, proximal/distal tubules |
| **Medulla** | Inner; renal pyramids (contain loops of Henle + collecting ducts) |
| Renal pyramids | 8–18 per kidney; apex (papilla) drains into minor calyx |
| Renal columns (of Bertin) | Cortical tissue between pyramids |
| Minor calyces | Surround each papilla |
| Major calyces | 2–3; formed by union of minor calyces |
| **Renal pelvis** | Funnel-shaped; drains into ureter at UPJ |

### Nephron (Functional Unit)
Glomerulus → Bowman's capsule → PCT → Loop of Henle → DCT → Collecting duct → Papilla → Calyx

## ⚙️ Function

| Function | Mechanism |
|----------|-----------|
| **Filtration** | ~125 mL/min GFR; filters 180 L/day |
| **Reabsorption** | ~99% water + solutes reabsorbed in tubules |
| **Secretion** | H⁺, K⁺, drugs, toxins secreted into tubule |
| **Blood pressure** | RAAS → angiotensin II + aldosterone |
| **Erythropoiesis** | EPO secreted by peritubular cells → stimulates RBC production |
| **Vitamin D activation** | 1α-hydroxylase converts 25-OH-D → 1,25-(OH)₂-D (calcitriol) |
| **Acid-base balance** | Bicarb reabsorption; H⁺ secretion |

## 🏥 Clinical Significance

| Condition | Key Points |
|-----------|-----------|
| **Renal cell carcinoma (RCC)** | Clear cell (most common); VHL mutation; "haematuria, flank pain, palpable mass"; treat with nephrectomy |
| **Nephrolithiasis** | Ca oxalate (most common); uric acid (radiolucent); passage causes ureteric colic (loin to groin) |
| **Hydronephrosis** | Obstruction → pelvi-calyceal dilation; causes: calculi, stricture, malignancy |
| **Acute kidney injury (AKI)** | Pre-renal / intrinsic (ATN, GN) / post-renal; RIFLE/KDIGO criteria |
| **Chronic kidney disease (CKD)** | GFR <60 for >3 months; complications: anaemia, CKD-MBD, hyperkalaemia |
| **Polycystic kidney disease** | ADPKD (PKD1/PKD2): bilateral cysts; HTN; berry aneurysm association |
| **Pyelonephritis** | Ascending UTI; flank pain, fever, pyuria; *E. coli* most common |
| **Renal transplant** | Placed in iliac fossa (heterotopic); renal artery anastomosed to external iliac artery |
