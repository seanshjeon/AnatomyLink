PROJECT OVERVIEW
================================================================================
anatomyLink is a comprehensive human anatomy learning application built with
Obsidian. It leverages Obsidian's linking features to create an interconnected
knowledge base where each anatomical structure displays information about its
components and how they connect to other structures.

This vault is designed for students and anyone interested in understanding
human anatomy through interactive connections and relationships.

SYSTEM REQUIREMENTS
================================================================================
- Obsidian (latest version recommended)
  Download from: https://obsidian.md/download
- Operating System: Windows, macOS, or Linux
- Storage Space: Minimum 50MB free space
- RAM: 2GB minimum (4GB recommended)

INSTALLATION INSTRUCTIONS
================================================================================

STEP 1: Install Obsidian
------------------------
1. Visit https://obsidian.md/download
2. Download the version for your operating system
3. Install Obsidian following the installer instructions
4. Launch Obsidian

STEP 2: Obtain the anatomyLink Vault
----------------------------------
The anatomyLink vault can be obtained through:
- GitHub repository (if available)
- Direct download from the source
- File sharing from the creator

Save the anatomyLink folder to your preferred location on your computer.

STEP 3: Open anatomyLink in Obsidian
------------------------------------
1. Launch Obsidian
2. Click "Open folder as vault" (or similar option)
3. Navigate to and select the anatomyLink folder
4. Confirm the vault opening
5. Obsidian will index all files - this may take a few moments

STEP 4: Explore the Vault
-------------------------
1. The left sidebar shows the folder structure with different body systems:
   - Circulatory System
   - Respiratory System
   - Endocrine System
   - And others...

2. Click on any system to explore its components
3. Follow the links between anatomical structures to see connections
4. Use the Graph View to visualize the relationships between structures

BASIC USAGE
================================================================================

NAVIGATING THE CONTENT
1. Open any file by clicking on it in the file explorer
2. Use [[wiki-style links]] to jump between related structures
3. Click the graph icon (⚛) to view the connection graph
4. Use Ctrl+F (or Cmd+F on Mac) to search for specific structures

CREATING YOUR OWN CONNECTIONS
1. While reading a file, you can add new links to connect structures
2. Format: [[Structure Name]] to link to another file
3. Links automatically create bidirectional connections

CUSTOMIZATION (OPTIONAL)
1. Modify CSS snippets for appearance changes
2. Install community plugins for enhanced functionality
3. Create personal notes linked to structures

TROUBLESHOOTING
================================================================================

Issue: Files don't open or show blank content
Solution:
- Check that all files have .md extension
- Restart Obsidian
- Verify the folder has read permissions

Issue: Links appear broken (red text)
Solution:
- The file may not exist yet
- Check spelling against actual filenames
- Create the missing file if needed

Issue: Graph view is slow
Solution:
- Close other applications to free RAM
- Disable unnecessary plugins
- Restart Obsidian

Issue: Files not syncing across devices
Solution:
- Use Obsidian Sync (paid subscription)
- Or use a cloud service (iCloud, Google Drive, Dropbox)
- Copy the vault folder to your preferred location
